1. Импортируем из Figma наши svg файли перед этим нажимаем Ctrl + Shift + O
2. Заходим на сайт https://icomoon.io/
3. Нажимаем IcoMoon App
4. Нажимаем на бургер справа, далее жмём Import to Set и выбираем наши файли
5. Убираем цвет у иконок цвет (делаем их одного цвета)
6. Вибираем их все инструментом выделения
7. Жмём внизу Generate Font
8. Меняем названия иконок (это названия будет нашим классом в дальнейшем)
9. Возле кнопки Download нажимаем шестерёнку и меняем Font Name на нужный
10. Жмём Download - скачиваеться архив
11. Закидуем файл style.css в папку scss/vendor/ при этом меняем название файла на iconfonts.scss
12. Закидуем файли из папки fonts в папку fonts/iconfonts/
13. Меняем путь к файлам из    /fonts/icons.eot?w5yy00    ===>    в     ../fonts/iconfonts/icons.eot?w5yy00    (добавляем 2 точки в начале, и папку /iconfonts) СКОПИРОВАТЬ!
14. Меняем [class^='icon-'],   на    [class^='icon-']::before,     (добавляем ::before)   
		   [class*=' icon-']		 [class*=' icon-']::before
15. Используем добавляя к элементам класс icon-названия класса в файле iconfonts.scss в самом низу (к примеру icon-search)
